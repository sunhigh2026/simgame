## 1. ゲーム概要

### 1.1 コンセプト

架空の国「ナホン」を舞台に、30歳の脱サラ主人公が貯金Ƴ500万で起業し、5年間の法人経営を体験するシミュレーションゲーム。法人設立、売上確保、人材管理、税金、節税といった「経営のリアル」を、カード選択とパラメータ設定のハイブリッドで体験する。

「リアルな日本の法人経営」の構造を再現しつつ、架空の国・架空の制度として免責を確保する。税率や制度名は現実と意図的にずらし、「構造はリアル、数字はフィクション」を基本方針とする。

## 4. ゲームフロー詳細

### 4.1 オープニング（会社設立フロー）

プレイヤーは以下を順番に選択・入力する。

**Step 1：導入テキスト** — ナホン国トーキョ区の描写。脱サラした主人公の状況説明。

**Step 2：業種選択** — Web制作 / カフェ開業 / EC物販 / コンサル・講師業の4択。それぞれ初期費用、売上の立ちやすさ、利益率、難易度が異なる。

**Step 3：会社形態** — 正社（設立費Ƴ25万、信用度+10）/ 略社（設立費Ƴ10万、信用度+0）の2択。

**Step 4：社名入力** — テキスト入力。変更不可。

**Step 5：資本金設定** — スライダーで設定。貯金Ƴ500万から設立費用・初期費用を引いた残りが上限。設定額がそのまま法人口座の初期残高になる。残りは個人の生活費。

**Step 6：決算月選択** — 3月 / 9月 / 12月の3択。設立月から遠い月を選ぶと1期目が長くなり免税期間を最大化できる、というTipsを税理士が教えてくれる。

**Step 7：代表給設定** — スライダーで月Ƴ0〜Ƴ60万の範囲で設定。会社の負担額（給与＋相互扶助金）と個人の手取り概算がリアルタイム表示される。年度内は変更不可。

### 4.2 月次ループ（メインゲーム）

毎月のループは以下の構造。

```
① イベントチェック
   → 該当月にイベントがあれば表示（テキスト選択型）
   → なければスキップ

② パッシブ収入の適用
   → Webサイト等の永続効果による自動売上の加算

③ 手札フェーズ
   → 山札から5枚引く
   → プレイヤーが2枚選択
   → 選ばなかった3枚は捨て札へ

④ カード効果の解決
   → コスト支払い → 成否判定 → 売上・効果の適用
   → 2枚分の結果をまとめて表示

⑤ 月末処理（自動）
   → 代表給 + 相互扶助金の支払い
   → 従業員の給料 + 相互扶助金の支払い
   → 永続的な固定費（家賃・積立等）の支払い
   → 基本運営経費（通信・雑費等）の支払い
   → 通帳残高の更新
   → 個人の手取りの加算

⑥ 資金ショートチェック
   → 法人口座がマイナスの場合、カウント+1
   → 2ヶ月連続でマイナスならゲームオーバー

⑦ 月の進行
   → 決算月の翌月なら決算処理へ
   → それ以外は次の月へ
```

### 4.3 決算処理（年1回）

決算月の翌月に発生する特別フェーズ。

**P/L（損益計算書）の表示：** 売上高、売上原価、売上総利益、販管費の内訳（代表給、人件費、相互扶助金、家賃、その他）、営業利益を表示。

**税金計算：** 繰越欠損金の適用→節税控除の適用→課税所得の確定→商益税（Ƴ800万以下14%、超22%）→市民割Ƴ7万（赤字でも発生）→事業割5%→取引税（3期目以降、前々期売上Ƴ1,000万超の場合）→合計税額の表示→節税効果の表示（対策なしとの差額）

**ポイント：** 赤字でも市民割Ƴ7万がかかることを強調表示。繰越欠損金で税金が減る体験。節税カードを使っていた場合、「対策なしならƳ○○だった」と比較表示。

**決算後：** 期の記録を保存。次の期の準備（代表給の再設定）へ。

### 4.4 期の切り替え（年度はじめ）

各期の冒頭で以下を行う。

代表給の再設定（前期の金額がデフォルト表示。変更可）。新カードのアンロック（期に応じて山札にカードが追加される）。デッキの再構築。

### 4.5 エンディング

5期目の決算終了後に表示。

**成績表：** 5年分のP/L推移（期ごとの売上・利益・税金）。累計売上、累計納税額、節税で浮いた額、最終法人口座残高、最終個人資産、従業員数、信用スコア。

**ランク判定：** S〜Dの5段階＋隠しランク。条件は後述（§8）。

**SNSシェア機能：** タイトル、業種、ランク、主要数値、ハッシュタグを含むテキストを生成。Web Share APIまたはクリップボードコピー。

## 5. ゲームメカニクス詳細

### 5.1 行動の二重構造

このゲームの中核設計。プレイヤーの行動は2種類に分かれる。

**カードで選ぶもの（＝何をするか）：** 営業する、広告を出す、人を雇う、設備を買う、節税対策をする、休む、等。毎月5枚の手札からから2枚を選ぶ。手札はランダムなので「やりたいことができない」状況が生まれる。この「ままならなさ」がリアルな経営感を再現する。

**自分で数字を決めるもの（＝いくらにするか）：** 代表給（年度はじめに設定、年度内変更不可）。従業員の給料（変更可、ただし下げると退職リスク上昇）。商品・サービスの価格（変更可、高くすると受注率低下）。仕入れ量（EC物販の場合）。

### 5.6 従業員システム

従業員には以下のパラメータがある。

|パラメータ|説明|
|---|---|
|name|名前|
|type|parttime / fulltime|
|salary|月給（プレイヤーが変更可能）|

**満足度の変動要因：** 給料を上げると上昇、下げると急降下。福利厚生導入で上昇。長期間昇給なしで徐々に低下。決算賞与で上昇。

**退職リスク：** 満足度40未満で退職リスクが発生。30未満で危険域。20未満でほぼ確実に退職。退職時は採用コスト（求人広告Ƴ30万＋面接時間）が再発生する。

### 5.7 税金計算ルール

#### 商益税（法人税に相当）

課税所得Ƴ800万以下の部分に14%、超の部分に22%。課税所得がゼロ以下なら0。

#### 市民割（法人住民税均等割に相当）

一律Ƴ70,000。赤字でも必ず発生する。

#### 事業割（法人事業税に相当）

課税所得の5%。赤字なら0。

#### 取引税（消費税に相当）

3期目以降、前々期の売上がƳ1,000万を超えた場合に課税事業者となる。簡易計算として売上 × 8% × 30%（みなし仕入率控除後）で概算。簡易方式/原則方式の選択イベントあり（3期目に発生）。

#### 繰越欠損金

赤字の期の損失額を翌期以降に繰り越し、将来の利益と相殺できる。

#### 節税控除

小商人積立、安全共済機構などの効果により課税所得を圧縮する。

## 14. 留意事項

### 14.1 法的リスクの回避

ゲーム冒頭に免責文を表示する。税率・制度名は現実と意図的にずらす。「税務相談」にあたる内容は避ける。架空の国・架空の制度である旨を一貫して示す。

### 14.2 ゲームバランスの方針

1期目は厳しく（生存の緊張感）。2期目で初めての黒字を体験させる。3期目で節税の効果を実感させる。4期目で試練（外的要因のリスク）。5期目で集大成の達成感。全体を通して「資金ショート」の恐怖と「黒字化」の快感のコントラストを最大化する。

### 14.3 教育的配慮

ゲームを通じて以下の概念を自然に学べることを目指す。法人と個人の財布の分離。役員報酬と所得税・社会保険のトレードオフ。赤字でもかかる固定税。繰越欠損金の仕組み。消費税（取引税）の課税事業者判定。節税と脱税の境界線。資金繰り（黒字倒産）の概念。人件費と従業員満足度のバランス。
